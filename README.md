# Library Management System

## Overview
This project implements a relational database for a Library Management System using PostgreSQL.

## Features
- 7 normalized tables
- Primary and foreign key relationships
- Sample data insertion
- SQL Views
- Analytical SQL Queries
- CAP Theorem discussion
- Query Performance Optimization

## Tools Used
- PostgreSQL (Supabase)
- dbdiagram.io
- GitHub

## Database Tables
- Members
- Authors
- Categories
- Books
- Librarians
- Loans
- BookReservations
